## Comparative Image Colorization using CNN and Conditional GAN

Built and compared a **CNN baseline** and a **Conditional GAN (cGAN)** for grayscale-to-color image translation on a subset of the **Places365** dataset.

### Highlights
- Implemented a **CNN baseline** for automatic image colorization.
- Built a **Conditional GAN** to generate more realistic and visually appealing colorized images.
- Improved GAN output quality using **perceptual loss**.
- Evaluated models using **image similarity metrics** and **qualitative visual comparison**.
